Tentang Kami

Kami Menyediakan Berbagai Macam Kebutuhan Digital Anda


Siapa Kami?
Nama Saya Fajar Utama. Saya adalah salah satu mahasiswa di Universitas Dian Nuswantoro. NIM Saya A11.2020.12488 dan kelas saya untuk mata kuliah Pemrograman Berbasis Web adalah A11.4304.




Kenapa Website Ini Dibuat?
Website ini ada karena untuk memenuhi Tugas Akhir Mata Kuliah Pemrograman Berbasis Web Universitas Dian Nuswantoro.





Apa Saja yang Ditawarkan Oleh Website Ini?
Website ini dibuat menggunakan HTML dan di poles menggunakan CSS dan sedikit Javascript. Sistem di website ini terintegrasi dengan database yang menggunakan PHP sebagai perantaranya. Mulai dari varian pembelian hingga transaksi yang masuk akan ditampung oleh database. Semua pemesanan dan direct link tidak benar-benar nyata, setiap pesanan yang masuk tidak akan diproses secara nyata. Hal ini dikarenakan website ini hanya untuk keperluan Tugas Akhir dan belum dapat diimplementasikan secara nyata. Semoga saja, website ini dapat terus dikembangkan agar menjadi website nyata yang dapat dipakai oleh konsumen. Aamiin...
